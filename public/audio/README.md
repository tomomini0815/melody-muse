# Audio Analysis Folder

Place your audio files here to be analyzed!
Supported formats: `.mp3`, `.wav`, `.ogg`, `.m4a`, `.flac`

## How to use

1.  **Place Files**: Drop your audio files into this folder (`public/audio`).
2.  **Run Analysis**: Open a terminal in the project root and run:
    ```bash
    py scripts/analyze_audio.py
    ```
3.  **View Results**: The script will generate `analysis_result.png` in the project root.
